# 🏘️ Projet 5 : Créez et utilisez une base de données immobilières avec SQL

### 📄 Contexte du projet

Création d'une **base de données relationnelle** **SQL** pour le réseau Laplace Immo. Structuration des données immobilières et extraction d'**indicateurs de performance** (**KPIs**) pour la direction régionale.

### 🎯 Objectifs de la mission
1. **Modélisation :** Conception du **schéma relationnel** (tables, clés primaires) et création de la base de données (**SGBD**).
2. **Documentation :** Élaboration du **dictionnaire des données** pour standardiser les variables du projet.
3. **Analyse SQL :** Écriture de **requêtes SQL** pour extraire les **KPIs** (ventes, locations, performance des agences).

### 🛠 Compétences Techniques (Hard Skills)
* **SQL :** Création de tables et rédaction de requêtes pour extraire des indicateurs précis.
* **Modélisation de données :** Conception d'un **schéma relationnel** et structuration des données immobilières.
* **SGBD :** Implémentation technique de la base de données pour stocker les informations des agences.
* **Documentation technique :** Élaboration d'un **dictionnaire des données** pour référencer les variables.

### 🧠 Compétences Générales (Soft Skills)
* **Compréhension du besoin métier :** Traduction des demandes de la direction (**KPIs**) en solutions techniques.
* **Rigueur :** Analyse précise des données pour garantir la fiabilité des indicateurs fournis.
* **Pédagogie :** Présentation de la méthodologie et des résultats via un support adapté.
